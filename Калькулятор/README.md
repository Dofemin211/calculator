# Калькулятор (HTML/CSS/JS)

Простий красивий калькулятор, який працює в браузері та містить:

- Адаптивний інтерфейс
- Темну і світлу тему
- Історію обчислень (до 20 останніх записів)
- Наукові функції: `sin`, `cos`, `tan`, `sqrt`, `ln`, `log`, `exp`, `±`
- Підтримку клавіатури
- Обробку точних чисел через `Big.js`
- Видалення `⌫`, очищення `AC`, оператори `+ - * / % ^`
- Анімації при натисканні та оновленнях історії

## Файли

- `index.html` - основний інтерфейс та логіка JavaScript
- `style.css` - стиль калькулятора

## Запуск

1. Відкрий `index.html` у браузері.
2. Натискай кнопки або користуйся клавіатурою.

## Клавіатурні комбінації

- 0-9, . — введення значень
- + - * / % ^ — операції
- Enter або = — обчислити
- Backspace — видалити один символ
- C — очистити усі
- s,c,t,l,n,e,r — функції sin, cos, tan, log, ln, exp, sqrt

## Налаштування теми

Натисни кнопку «Темна тема / Світла тема» для зміни стилю.

## Розвиток

Можливі покращення:

- Збереження історії в `localStorage`.
- Додаткові математичні функції (`x^2`, `1/x`, `rad/deg`).
- Підтримка drag&drop та темної/світлої анімації.
